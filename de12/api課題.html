API課題

自分が一番使う最寄り駅のダイヤを検索するソースコードを書きました。
本当はとこ浜駅なども追加したかったのですが、時間がなかったのでとりあえず久里浜駅西堀作りました。
時間がある時にexcelで作ったcsvファイルにアップデートしていきたいと考えています。

今回つくったソースコード


    import pandas as pd

#cvsファイルの読み込み
df = pd.read_csv("de12/pp/京急久里浜.csv")
 
station_name="久里浜"
start_time="8:00"
end_time="9:00"

filtered_df=df[(df["駅名"]== station_name) &
(df["時刻"]>=start_time)&
(df["時刻"]<=end_time)]

if not filtered_df.empty:
    print(filtered_df)
else:
    print("該当するダイヤがありません。")

感想

今回の課題を制作する上で、ダイヤ検索のソースとしてのファイルをcsv形式で作ったのですが、
そのためにいちいちアナログに打ち込んで作ったのでクソめんどくさかったです。
もっといい方法がないか模索する必要がありました。
また、今回自分が作ったものより便利なものがネットの海にはたくさんあるため、ソースコード書く以前に
アイデアを生み出すことのほうが肝心だと気づきました。
